type num 
type matrix 
val num_of : string -> num

num_of s is the numeric value matched from string s. Fails if the string does not match a numeric value. Ex: num_of "5.0" = Float 5.0

val matrix_of : string -> matrix

matrix_of s is the matrix matched by string s. Fails if the rows of the matrix have different lengths or if the type of elements are not numeric Ex: matrix_of "1 2 3; 1 2 3" = [Int 1; Int 2; Int 3]; [Int 1; Int 2; Int 3]

type unaryop =
| Transpose
| Inverse
| Det
| Rowspace
| Nullspace
| Colspace
| Reduce
| QRFactor
| LuDecomp

unaryop is a type that classifies operations with one argument

type binaryop =
| Add
| Sub
| Div
| Mul
| Dot
| Solve
| Scale
| ChangeBasis
| OrthProject
| DistToBasis
| Decomp

binaryop is a type that classifies operations with two arguments

type expr =
| Matrix of matrix
| Num of num
| Unary of (unaryop * expr)
| Binary of (expr * binaryop * expr)
| List of expr list

expr is a type that classifies operations on matricies and numbers

type typ =
| TMatrix
| TNum
| TList of typ

typ is a type that classfiies the type of an expr

val typecheck : expr -> typ

typecheck e is the type of the expr. Fails if the type is incoherent

val eval : expr -> expr

eval e is the evaluated expression (value) from e. All expressions step to either Matrix m or Num n eventually. Will not during steps since typecheck is applied at the beginning.

val format_expr : Format.formatter -> expr -> unit

format_expr e is the printing function for the expr
